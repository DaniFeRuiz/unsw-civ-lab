# UNSW CIV Lab Quarto starter

1. Put these files in the root of your Quarto project folder.
2. Keep the image at `assets/images/civlab-v2.png`.
3. Run `quarto preview` while editing.
4. Run `quarto render` before committing to GitHub.
5. For the simple Cloudflare Pages deployment, commit the `_site` folder too. Do not add `_site/` to `.gitignore` yet.
6. In Cloudflare Pages, use build command `exit 0` and build output directory `_site`.
